# apple-music-download

Download Link: https://www.icloud.com/shortcuts/9966f7b0df4b4688963c1096f9220264

How to use:
1. Downlod and install Shortcuts app on your iOS device.
2. Run any built-in shortcut.
3. Enable unknown shortcuts by going to Settings >Shortcuts > Allow Untrusted Shortcuts.
4. Follow the link above and run the script to download media easily in One click.

This script ease the process of downloading music from your iTunes cloud. Apple Music subscription required 

Working:
This script pull all the songs that are stored on cloud in your iTunes and make a list of all songs. 
Then it create a playlist out of that list. Now you can simply go to Music app on your iOS device and go to playlists.
There you'll see a playlist called "OneClick Music". Open that playlist and click on Download icon in top left corner.
It'll download the music from your cloud storage to iphone.
